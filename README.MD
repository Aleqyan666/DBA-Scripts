# MS SQL Server DBA Scripts

This repository contains Microsoft SQL Server scripts written for database administration tasks.

The goal of this repository is to collect useful utilities for SQL Server operations such as database maintenance, monitoring, storage management, and troubleshooting.

The repository is currently in its early stages and will gradually include more scripts and tools used in day-to-day DBA work.
